# Sistema de Control de Fichajes (Java)

Aplicación de consola desarrollada en Java para gestionar el fichaje de trabajadores.

El sistema permite a administradores gestionar empleados y consultar registros de jornada,
mientras que los trabajadores pueden registrar su entrada y salida.

## Funcionalidades

- Login de usuarios
- Roles: Administrador y Trabajador
- Alta y baja de trabajadores
- Consulta de trabajadores
- Cambio de contraseña
- Registro de fichajes
- Visualización de registros de jornada
- Persistencia de datos en archivos

- ## Tecnologías

- Java
- Programación Orientada a Objetos (POO)
- Manejo de archivos
- Arquitectura básica MVC

- ## Cómo ejecutar

1. Clonar el repositorio
git clone https://github.com/JoseeAlonso/Administracion-de-Trabajadores-y-Fichajes-1DAW-Jose-Alonso.git

3. Abrir el proyecto en IntelliJ, Visual Studio Code o cualquier IDE compatible con lenguaje JAVA
